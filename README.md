## 🏁 Getting Started <a name = "getting_started"></a>

- clone project
  ```
  git clone https://github.com/harshraj8843/tic-tac-toe-react
  ```
  ```
  cd tic-tac-toe-react/
  ```
- Install packages
  ```
  yarn
  ```
- Start development server
  ```
  yarn start
  ```
